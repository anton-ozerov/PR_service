# Запуск
```
cp .env.example .env
docker compose up --build
```

# Тестирование
По http://0.0.0.0:8000/docs доступна документация. Если не меняли `.env` файл, то логин - `admin`, пароль - `admin_password`

Для тестов были созданы данные через миграции:
3 команды с названиями: First, Second, Third
4 is_active=True пользователя:
   * админ - логин `admin`, пароль `admin` (если не меняли в `.env`), команда 1
   * юзер 1 - логин `user1`, пароль `userpass`, команда 1
   * юзер 2 - логин `user2`, пароль `userpass`, команда 1
   * юзер 3 - логин `user3`, пароль `userpass`, команда 2
